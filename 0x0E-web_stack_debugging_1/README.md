# 0x0E-web_stack_debugging_1
This repository contains solutions to debugging tasks related to web stack configuration and troubleshooting. The tasks are part of the alx-system_engineering-devops curriculum.

Task 0: Nginx likes port 80
Problem Description
The goal of this task is to troubleshoot and fix an issue where the Nginx web server is not listening on port 80 of the server's active IPv4 IPs.

Solution
The Bash script 0-nginx_likes_port_80 addresses the issue by updating the Nginx configuration to listen on port 80 and then restarting the Nginx service.


Task 1: Make it sweet and short
Problem Description
The objective of this task is to optimize the Bash script used in Task 0, making it short and sweet. The script should be no more than 5 lines long and should ensure that Nginx is not running after execution.

Solution
The Bash script 1-debugging_made_short accomplishes the task by using minimal lines to update the Nginx configuration and restart the service.


Requirements
Ubuntu operating system
Nginx web server installed
